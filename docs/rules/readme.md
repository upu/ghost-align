# README 運用ガイド

このリポジトリの利用者向けドキュメント [`README.md`](../../README.md)（英語・正） / [`README.ja.md`](../../README.ja.md)（日本語）の書き方・運用方針をまとめたコントリビューター向け資料です。

## 役割

`README.md` は VS Code Marketplace の拡張機能ページとして公開される、利用者向けドキュメントです。`README.ja.md` は日本語版で、`README.md` へのリンクで相互参照します。

## 二言語運用（英語が正）

[CHANGELOG](changelog.md) と同じく、**英語版 `README.md` が正**で、日本語版 `README.ja.md` を併記します。

## 書く / 書かない の基準

- **書く**: 機能紹介・インストール・使い方・設定など、利用者向けの内容。
- **書かない**: 開発者向けの記述（ビルド・デバッグ・コントリビュート手順、内部実装メモなど）。それらは [`docs/CONTRIBUTING.md`](../CONTRIBUTING.md) に置きます。

## 同期

一方を変更したら、原則として同じ変更をもう一方にも反映し、内容を同期させます（構成・セクション・設定表などが2ファイル間でズレないようにする）。文言そのものの翻訳まで機械的に一致させる必要はありませんが、機能・設定・手順の記述内容は揃えます。
