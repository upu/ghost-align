# README.md / README.ja.md は利用者向け

`README.md`（英語、VS Code Marketplace の拡張機能ページとして公開）と `README.ja.md`（日本語版、`README.md` へのリンクで相互参照）は、どちらも利用者向けドキュメントです。

- 開発者向けの記述（ビルド・デバッグ・コントリビュート手順、内部実装メモなど）をどちらにも書かないこと。それらは `docs/CONTRIBUTING.md` に置く。
- どちらにも利用者向けの内容（機能紹介・インストール・使い方・設定）だけを置く。
- 一方を変更したら、原則として同じ変更をもう一方にも反映し、内容を同期させる（構成・セクション・設定表などが2ファイル間でズレないようにする）。文言そのものの翻訳まで機械的に一致させる必要はないが、機能・設定・手順の記述内容は揃えること。
